# Quiz Management System

A full-stack application for managing and taking quizzes.

## Project Structure

- `quiz-backend/`: FastAPI backend with PostgreSQL.
- `quiz-frontend/`: React frontend with Vite and TypeScript.

## Quick Start
### Backend

1.  Navigate to `quiz-backend`.
2.  Set up environment: `python3 -m venv venv && source venv/bin/activate`
3.  Install dependencies: `pip install -r requirements.txt`
4.  Configure `.env` (see README in backend folder).
5.  Run: `python main.py`

### Frontend

1.  Navigate to `quiz-frontend`.
2.  Install dependencies: `npm install`
3.  Run: `npm run dev`

See specific README files in each directory for more details.
