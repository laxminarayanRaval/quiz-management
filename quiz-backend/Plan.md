## Plan

1. Quiz management :
- Backend :  Python (FastAPI)
- Database : NeonDB
- Frontend : React
- UI framework : Tailwind CSS


2. Admin Panel :
- Should able to create, update and delete quizzes
- Should able to add, edit and delete questions and answers
- questions of various types : 
    - Single choice (radio)
    - Multiple choice (checkbox)
    - True or False (radio)
    - Fill in the blank (text)

3. User Panel :
- Should able to take quizzes
- Should able to view results
- Should able to view  leaderboard

- High Level Design : 
    - Admin Panel : 
        Schema :
          - Quizzes
          - Questions
          - Answers
          - Results
    - User Panel :
        Schema :
          - Users
          - Results
          - Leaderboard
